What's Inside

⛏

Proof-of-Work Mining

Mine blocks with configurable difficulty. Watch the nonce increment until a valid hash is found.

🔑

Digital Signatures

Ed25519 key pairs sign every transaction. Cryptographic proof that you authorized the transfer.

💰

Native Coin

Miners earn block rewards. Send coins between wallets. Track balances across the chain.

🔗

Chain Validation

Every block links to the last. Tamper with one and the entire chain breaks — by design.

🌐

Multi-Node Network

Run multiple nodes locally. They share blocks, validate chains, and reach consensus.

⚙

Fully Customizable

Change the coin name, mining difficulty, block reward, and block size from a single config file.

How It Works

1

Create a Wallet

Generate an Ed25519 key pair. Your public key is your address. Your private key signs transactions.

2

Send a Transaction

Sign a transfer with your private key. It enters the pending pool, waiting to be included in a block.

3

Mine a Block

Pick up pending transactions, add a coinbase reward, and find a nonce that produces a hash with enough leading zeros.

4

Chain Grows

The new block links to the previous one via its hash. Any tampering would invalidate every block that follows.

5

Network Consensus

Nodes share their chains. The longest valid chain wins — this is how the network agrees on truth.
